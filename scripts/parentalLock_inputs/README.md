# Toggles Parental Lock On/off on HDMI inputs and verifies

Requirements:
* Developer options enabled in settings
* Parental lock OFF initially
* Parental lock code should be “0000”

What it does:
* Navigates to Parental Lock under settings menu.
* Toggles Parental lock off/on.
* Tunes back to input after each toggle to verify lock/unlock.
